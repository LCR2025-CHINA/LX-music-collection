
# lx-music-Source of music

洛雪音乐音源存储仓库

## 项目简介

本仓库专门用于收集、整理和存储洛雪音乐客户端的音源文件，为洛雪音乐提供稳定可靠的音乐源支持。

- **音源的可用性需要大家自行尝试**，不保证所有音源的稳定性
- 音源质量可能因平台策略调整而变化
- 请遵守相关音乐平台的使用条款和版权规定

## 使用方法

### 手动导入音源
1. 下载所需的音源文件(.js)
2. 打开洛雪音乐客户端
3. 进入设置 → 音乐源管理
4. 点击"导入"选择下载的音源文件
5. 测试音源可用性

### 自动更新
- 支持通过音源订阅链接自动更新
- 在设置中配置自动更新频率
